description=
description=[h1]Description[/h1]
description=The Log Extender mod is designed for Project Zomboid servers and adds log entries that helps administering the server.
description=The mod allows you to save information about the levels of perk character, the time of his survival, the number of killed zombies and other
description=information about the character, which, as you know, is stored on users' local computers. The mod also fixes several game bugs that are related
description=with logging the actions of players. For example, a record is added about taking a generator from the ground. You can read more about this bugreport
description=on [url=https://theindiestone.com/forums/index.php?/topic/25683-nothing-will-be-written-to-the-log-if-you-take-generator-from-the-ground/]theindiestone.com forum[/url].
description=
description=[h1]How to Use[/h1]
description=The mod does not work in single player mode. Therefore, it must be installed on a dedicated Project Zomboid server.
description=
description=[h1]Warning[/h1]
description=Mod Log Extender is under development and is being tested on the server [url=https://last-day.wargm.ru]Last Day[/url]. You can join our server or use the mod on your own server.
description=If you think you have found a bug, write about it in the [url=https://steamcommunity.com/workshop/filedetails/discussion/1844524972/1638668751263547005/]bug reporting topic[/url]
description=the Steam workshop or create issue in [url=https://github.com/game-servers/pz-mod-log-extender]github repository[/url].
